# PickPDF

PDFから必要なページだけをスワイプで選んで、1つのファイルにまとめてダウンロードできるWebアプリです。

## Demo

GitHub Pagesで公開後、以下のURLでアクセスできます：

```
https://<your-username>.github.io/PickPDF/
```

## Features

- **PDFアップロード** - ファイル選択またはドラッグ＆ドロップでPDFを読み込み
- **スワイプで選択** - 右スワイプで「残す」、左スワイプで「削除」
- **ボタン操作** - スワイプの代わりにボタンでも操作可能
- **キーボード操作** - `→`キーで残す、`←`キーで削除、`Z`で戻す、`Space`で拡大
- **拡大表示** - ページを拡大・縮小して細部を確認可能
- **元に戻す** - 間違えても1つ前の操作に戻せる
- **PDFダウンロード** - 選んだページだけを1つのPDFファイルとしてダウンロード
- **完全クライアントサイド** - サーバーへのアップロード不要、ブラウザ内で完結

## Usage

1. PDFファイルをアップロード
2. 各ページを確認し、右スワイプ（または右矢印/KETEPボタン）で残す
3. 不要なページは左スワイプ（または左矢印/DROPボタン）で削除
4. 全ページの選択が完了したら「PDFをダウンロード」ボタンでダウンロード

## Tech Stack

- **PDF.js** - PDFの描画
- **pdf-lib** - PDFの編集・生成
- **Vanilla JS** - フレームワーク不使用、単一HTMLファイル

## Setup (GitHub Pages)

1. このリポジトリをGitHubにプッシュ
2. **Settings** > **Pages** > Source を `main` ブランチに設定
3. 公開URLにアクセス
